# Review Changes

Review code changes in a feature branch before merging. Categorizes issues by severity (Critical, Major, Minor) with actionable solutions.

## Installation

```bash
npx skills add artmann/review-changes
```

## Usage

The skill activates automatically when you ask your AI agent to review changes, check a PR, or audit code before merge.

Example prompts:

- "Review my changes before I merge"
- "Check this branch for issues"
- "Audit the code in this PR"
